# Markdown 示例

这是一个展示Markdown功能的示例文件。

## 1. 标题
Markdown支持多级标题，使用`#`符号表示：

## 2. 段落
段落之间需要空一行，例如：
这是第一段。

这是第二段。

## 3. 列表
### 无序列表
使用`-`、`*`或`+`表示无序列表：
- 项目1
- 项目2
  - 子项目2.1
  - 子项目2.2
* 项目3
+ 项目4

### 有序列表
使用数字加点表示有序列表：
1. 第一项
2. 第二项
   1. 子项2.1
   2. 子项2.2
3. 第三项

## 4. 代码块
### 行内代码
使用反引号表示行内代码，例如：`print("Hello, World!")`。

### 多行代码块
使用三个反引号包裹多行代码，并指定语言（可选）：
```python
def hello_world():
    print("Hello, World!")
```

## 5. 引用
使用`>`表示引用：
> 这是一个引用示例。
> 引用可以跨多行。

## 6. 链接和图片
### 链接
使用`[文本](URL)`表示链接，例如：[Google](https://www.google.com)。

### 图片
使用`![替代文本](图片URL)`表示图片，例如：
![Markdown Logo](https://markdown-here.com/img/icon256.png)

## 7. 表格
使用`|`和`-`创建表格：
| 序号 | 名称   | 描述       |
| ---- | ------ | ---------- |
| 1    | 项目1  | 描述1      |
| 2    | 项目2  | 描述2      |
| 3    | 项目3  | 描述3      |

## 8. 强调
使用`*`或`_`表示斜体，`**`或`__`表示加粗：
- *斜体文本*
- **加粗文本**
- **_斜体加粗文本_**

## 9. 分割线
使用三个或以上的`-`、`*`或`_`表示分割线：

---

## 10. 任务列表
使用`- [ ]`表示未完成任务，`- [x]`表示已完成任务：
- [x] 任务1
- [ ] 任务2
- [ ] 任务3
